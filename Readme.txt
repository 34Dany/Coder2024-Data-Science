## General Info

El proyecto intenta describir, la relación existente entre las diversas variables que afectan al hecho delictivo 
"Robo/Hurto" de los vehículos, con el fin de poder obtener insights del análisis de datos y mediante un modelo de regresión lineal, 
para entender cómo afectan estas variables el hecho delictivo mencionado y asi una compañía de seguros pueda obtener información de como adecuar sus costos / acciones / suscripciones de acuerdo a este análisis.
